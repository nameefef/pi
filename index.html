🍓 树莓派 OS 使用教程大全

从零开始 · 手把手教学 · 小白也能看懂

1. 什么是树莓派？

树莓派（Raspberry Pi） 是一台 只有信用卡大小的电脑。

它可以做普通电脑能做的事：上网、看视频、写文档、编程。但是它很小、很便宜（几百块钱），而且还带了一排叫 GPIO 的小针脚，可以连接灯泡、马达、传感器之类的电子元件。

💡 一句话理解：树莓派 = 一台迷你 Linux 电脑 + 可以控制硬件的接口。

目前常见的型号：树莓派 4B（最普及）、树莓派 5（最新款）、树莓派 Zero 2W（超小型）。

2. 你需要准备什么？

要把树莓派用起来，你需要这些东西：

配件说明必须吗？
树莓派主板就是这台小电脑本身✅ 必须
SD 卡（至少 16GB）就像电脑的硬盘，装系统和文件用✅ 必须
5V 电源适配器（Type-C）给树莓派供电，建议买官方电源✅ 必须
显示器（带 HDMI）电视或电脑显示器都可以⚠️ 推荐
HDMI 线连接树莓派和显示器⚠️ 推荐
键盘和鼠标USB 接口的就可以⚠️ 推荐
SD 卡读卡器用来把系统烧录到 SD 卡⚠️ 需要
散热片 + 风扇防止树莓派过热🔧 建议
💡 不想买显示器也能用：你可以用"无头模式"——只给树莓派通电和联网，然后用你的电脑远程连上去。

3. 烧录系统到 SD 卡

"烧录"就是把操作系统装到 SD 卡上，就像给新电脑装 Windows 一样。

3.1 下载 Raspberry Pi Imager

官方提供了一个超简单的工具叫 Raspberry Pi Imager。

  1. 打开浏览器，去官网：https://www.raspberrypi.com/software/
  2. 下载适合你电脑的版本（Windows / Mac / Linux）
  3. 安装并打开这个软件

3.2 烧录步骤

1 把 SD 卡插进电脑的读卡器
2 打开 Raspberry Pi Imager
3 点击 "选择操作系统" → 选 "Raspberry Pi OS (32-bit)"（新手推荐带桌面版）
4 点击 "选择存储卡" → 选你的 SD 卡
5 🔥 关键一步：点击右下角的齿轮图标 ⚙️（设置），开启：
✅ 设置用户名和密码
✅ 设置 WiFi
✅ 开启 SSH
✅ 设置地区为 Asia/Shanghai，键盘布局为 us
6 点击 "写入" → 等进度条跑完（大概 5-10 分钟）
7 烧录完成后弹出 SD 卡，插到树莓派上
⚠️ 重要提醒：烧录会清空 SD 卡上所有数据！如果有重要文件先备份。

3.3 其他操作系统选择

  • Raspberry Pi OS (32-bit) — 官方系统，最稳定，新手首选
  • Raspberry Pi OS Lite — 没有桌面，只有命令行，适合服务器
  • Ubuntu Server / Desktop — 喜欢 Ubuntu 的可以用
  • LibreELEC / Kodi — 当电视盒子用
  • RetroPie — 当游戏机用

4. 第一次开机设置

  1. 把 SD 卡插到树莓派背面
  2. 插上键盘、鼠标、显示器（HDMI 线）
  3. 插上电源线（Type-C）—— 通电自动开机！

4.1 开机过程

你会先看到屏幕上有彩色方块闪过，然后是一堆文字滚动（那是 Linux 在启动），最后会出现桌面。

💡 第一次启动比较慢（30秒到1分钟），以后会快一些。耐心等待！

4.2 初始设置（第一次进系统）

  • 登录：默认是 pi / raspberry
  • 更改密码：点左上角菜单 → Preferences → Raspberry Pi Configuration → 改密码
  • 连接 WiFi：点击右上角网络图标
  • 更新系统：打开终端，输入：
sudo apt update            # 检查更新
sudo apt upgrade -y        # 安装更新
🚨 千万不要：在没点关机的情况下直接拔电源！关机方法：sudo shutdown now

4.3 安装中文输入法（新手必看）

树莓派系统默认只有英文输入，要打中文得自己装输入法。下面教你装 fcitx + 搜狗拼音，装完就能跟 Windows 一样用拼音打字了。

方法一：用 Pi-Apps 装（最简单）

  1. 先装好 Pi-Apps（看第23章）
  2. 打开 Pi-Apps → 搜 "Chinese" 或 "Sogou"
  3. 找到 Chinese Input (fcitx + Sogou) → 点 Install
  4. 装完后重启树莓派
  5. 按 Ctrl + 空格 切换中英文输入！

方法二：手动安装 fcitx + 搜狗拼音

# 第一步：安装 fcitx 输入法框架
sudo apt update
sudo apt install fcitx fcitx-pinyin fcitx-configtool -y

# 第二步：下载搜狗拼音 for ARM（树莓派版）
wget https://ime.sogou.com/dl/20230103134913/sogoupinyin_4.2.1.145_arm64.deb -O sogou.deb

# 第三步：安装搜狗拼音
sudo dpkg -i sogou.deb
sudo apt install -f -y          # 修复缺的依赖

# 第四步：设置 fcitx 为默认输入法
sudo nano ~/.profile

在文件末尾添加：

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
5 重启树莓派：sudo reboot
6 重启后按 Ctrl + 空格 切换中英文输入，点右上角键盘图标可以设置

方法三：只装拼音输入法（不装搜狗，省空间）

不想装搜狗，用系统自带的拼音也行：

sudo apt install fcitx fcitx-googlepinyin -y
sudo reboot
💡 输入法常用快捷键：
Ctrl + 空格 — 切换中/英文
Ctrl + Shift — 切换输入法（如果你装了多个）
Shift — 临时切换中/英文（打字时最常用）
右上角键盘图标 — 点它打开输入法设置

终端（Terminal） 打开方式：点 🍓 → Accessories → Terminal。

📂 文件和文件夹操作

pwd                       # 查看当前在哪个文件夹

📝 sudo 是什么意思？

📝 怎么知道文件/文件夹的完整路径？

这是新手最常卡住的地方——你知道要 cd，但不知道写什么路径。下面教你三种方法：

方法一：用 pwd 看当前目录（最简单）

pwd               # 显示你当前在哪个文件夹
                     # 输出类似：/home/pi/Desktop

方法二：用 realpath 看文件的完整路径

realpath 文件名    # 显示这个文件的完整路径
                     # 比如：/home/pi/Downloads/project.zip

realpath 文件夹名  # 显示文件夹的完整路径
                     # 比如：/home/pi/pi-apps

方法三：在文件管理器里看（不用打字）

  • 打开桌面的 File Manager（文件管理器）
  • 找到你的文件/文件夹
  • 右键 → 点 Properties（属性）
  • 或者直接看窗口顶部的地址栏，那里就显示着完整路径
💡 小技巧：在文件管理器里打开文件夹，然后右键空白处 → Open in Terminal，终端就会自动 cd 到这个路径，不用自己打字！
  • nano 文件名 → 打开文件
  • Ctrl+O → 保存后回车确认
  • Ctrl+X → 退出

8. 连接 WiFi 和网络

8.1 桌面版连接 WiFi

  1. 点击右上角网络图标
  2. 找到你的 WiFi 名字，输入密码

8.2 命令行连接 WiFi

sudo nano /etc/wpa_supplicant/wpa_supplicant.conf

文件末尾添加：

network={
    ssid="你的WiFi名称"
    psk="你的WiFi密码"
    key_mgmt=WPA-PSK
}
sudo wpa_cli -i wlan0 reconfigure

8.3 查看 IP 地址

hostname -I     # 查看 IP

9. SSH 远程连接

SSH 让你远程操作树莓派，不用接显示器。

9.1 开启 SSH

终端输入：sudo systemctl enable ssh && sudo systemctl start ssh

9.2 从 Windows 连接（用 PuTTY）

  1. 下载 PuTTY：https://www.putty.org/
  2. Host Name 填树莓派 IP，端口 22，点 Open
  3. 输入用户名密码即可

9.3 从 Mac / Linux 连接

ssh pi@192.168.1.100

9.4 如何知道树莓派的 IP？

有显示器就输 hostname -I，没显示器就查路由器管理页面。

💡 试试：ssh pi@raspberrypi.local

10. VNC 远程桌面

SSH 只能看到文字，VNC 能看到树莓派的桌面。

10.1 开启 VNC

sudo systemctl enable vncserver-x11-serviced
sudo systemctl start vncserver-x11-serviced

10.2 连接

下载 VNC Viewer，输入树莓派 IP 连接即可。

11. GPIO 引脚入门

GPIO 是树莓派最牛的特色——控制电子设备。

11.1 引脚图

终端输入 pinout 查看引脚图。

💡 重点：5V / 3.3V 供电、GND 接地、GPIO 编号可编程控制。

11.2 用 Python 控制 GPIO（点亮 LED）

#!/usr/bin/env python3
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

for i in range(5):
    GPIO.output(17, GPIO.HIGH)
    time.sleep(1)
    GPIO.output(17, GPIO.LOW)
    time.sleep(1)

GPIO.cleanup()

运行：python3 blink.py

🚨 注意：不要给 GPIO 输入超过 3.3V！不要直接把 LED 接到 5V！一定要串电阻！

11.3 常用 GPIO 库

  • RPi.GPIO — 最基础
  • gpiozero — 更简单，适合新手
  • pigpio — 功能强大

13. 如何解压文件

⚠️ 第一步永远先 cd：在终端里解压文件之前，先用 cd 命令进入压缩包所在的文件夹！ 比如你的 project.zip 在桌面上，就先输入 cd ~/Desktop 再解压。 
cd ~/Desktop          # 先进入桌面文件夹
unzip project.zip     # 再解压
格式解压命令
.zipunzip 文件.zip
.tar.gztar -xzf 文件.tar.gz
.tar.bz2tar -xjf 文件.tar.bz2
.tartar -xf 文件.tar
.gzgunzip 文件.gz
.7z7z x 文件.7z（需装 p7zip）

压缩文件

zip -r 输出.zip 文件夹/           # 压缩成 .zip
tar -czf 输出.tar.gz 文件夹/      # 压缩成 .tar.gz
💡 中文乱码：unzip -O GBK 文件名.zip

14. 如何运行 .sh 文件

⚠️ 第一步永远先 cd：在终端运行 .sh 文件之前，先用 cd 命令进入 .sh 文件所在的文件夹！ 比如你的 install.sh 在 /home/pi/pi-apps/ 里，就先用 cd 进去，再运行它。 
cd ~/pi-apps             # 先进入文件所在文件夹
chmod +x install.sh     # 给权限
./install.sh             # 再运行
偷懒方法：也可以用 bash 带上完整路径，这样不用 cd：bash /home/pi/pi-apps/install.sh
1 先 cd 到文件所在的文件夹：cd 文件夹路径
2 给权限：chmod +x 文件名.sh
3 运行：./文件名.sh

或者一步到位，不用 cd、不用给权限：

bash 文件名.sh   # 但文件必须在当前文件夹
bash /完整的/路径/文件名.sh   # 写完整路径，任何位置都能跑
💡 新手最推荐：记得 cd 到文件夹后再用 bash 文件名.sh，不用改权限，最安全。

15. 树莓派该安装什么型号的软件？

💡 树莓派的 CPU 是 ARM 架构，普通电脑是 x86 架构。两个不通用！
下载时选 arm64 或 aarch64 ✅，amd64 ❌ 不能用。
字样树莓派能用？
arm64 / aarch64✅ 能用
armhf / armv7l✅ 也能用
amd64 / x86_64❌ 不能用
i386 / i686❌ 不能用

16. 系统设置与配置

16.1 raspi-config

sudo raspi-config

常用设置：改密码、开 SSH/VNC/I2C/Spi、SD 卡扩容时区等。

⚠️ SD 卡扩容：Advanced Options → Expand Filesystem → 重启生效

16.2 改主机名

sudo raspi-config → 1 System Options → Hostname

16.3 设置静态 IP

sudo nano /etc/dhcpcd.conf
文件末尾加： 
interface wlan0
static ip_address=192.168.1.100/24
static routers=192.168.1.1
static domain_name_servers=8.8.8.8

17. 常见问题解决

  • 开不了机？检查 SD 卡、电源（5V/3A）、HDMI 线
  • 太烫了？正常 40-80°C，超 85°C 降频，加散热片
  • SSH 连不上？同 WiFi？SSH 开了没？IP 输对了？
  • SD 卡满了？sudo apt autoremove + sudo apt autoclean
  • 关机：sudo shutdown -h now，等绿灯灭再拔电

18. 搭建 NAS 家庭网络存储

💡 一句话：树莓派插大硬盘，变成你家专属"百度网盘"。

18.1 用 Samba 文件共享

1 插硬盘 → lsblk 查看（一般是 /dev/sda1）
2 挂载：sudo mount /dev/sda1 /media/usb
3 安装 Samba：sudo apt install samba samba-common-bin -y
4 sudo nano /etc/samba/smb.conf → 末尾加：
[MyNAS]
   path = /media/usb
   writeable = yes
   guest ok = no
   create mask = 0777
   directory mask = 0777
5 sudo smbpasswd -a pi → 重启：sudo systemctl restart smbd

18.2 从其他设备访问

Windows：\\树莓派IP\MyNAS   Mac：smb://树莓派IP/MyNAS

22.2 Smart Mirror 魔法镜

25. 如何知道文件/文件夹的路径？

这是新手最常卡住的地方——你知道要 cd，但那个路径到底长什么样？去哪儿看？下面教你四种方法，总有一种适合你。

25.1 方法一：用 pwd 看当前目录（最简单）

打开终端，输入 pwd，它直接告诉你此时此刻你在哪个文件夹：

pwd
/home/pi/Desktop   # 原来我在桌面！

pwd = Print Working Directory（打印当前工作目录）。每次你迷路了，就输入 pwd，它告诉你你在哪。

25.2 方法二：用 realpath 看任意文件/文件夹的路径

不管文件在哪个犄角旮旯，realpath 都能告诉你它的完整路径：

realpath project.zip
/home/pi/Desktop/project.zip

realpath pi-apps    # 文件夹也行
/home/pi/pi-apps

realpath 123.txt
/home/pi/Downloads/123.txt   # 原来它在 Downloads 里！

有了完整路径，你就能直接 cd /home/pi/Downloads 跳过去了。

25.3 方法三：在文件管理器里看（树莓派版"访达"）

不想打字？完全可以在图形界面里操作。

方法A — 直接看窗口顶部的地址栏：

  1. 打开桌面的 File Manager（文件管理器，在桌面左上角 🍓 → Accessories → File Manager）
  2. 导航到你要的文件夹
  3. 看窗口顶部，那里就显示着当前路径，比如 /home/pi/Downloads
  4. 点一下路径的文字，它会变成可编辑状态，按 Ctrl+C 就能复制路径

方法B — 右键看属性：

  1. 在文件管理器里找到一个文件或文件夹
  2. 右键 → 点 Properties（属性）
  3. 弹出的窗口里会显示 Parent Folder（父文件夹），就是它的完整路径

方法C — 最偷懒的大招（强烈推荐！）：

  1. 在文件管理器里打开你想要的文件夹
  2. 在 空白处右键 → 选 Open in Terminal
  3. 终端自动打开，而且已经自动 cd 到这个文件夹了！
  4. 输入 pwd 看一眼，路径已经在那里了，直接输命令操作就行
💡 总结这三种方法：
• 只想看路径 → 看窗口顶部的地址栏
• 想复制路径 → 点地址栏 → Ctrl+C
• 想直接操作 → 右键 → Open in Terminal

25.4 方法四：用 find 命令搜文件（不知道文件在哪的时候用）

你只记得文件名，但不知道它在哪个文件夹里？用 find 搜：

find ~ -name "project.zip"          # 在 home 里搜 project.zip
/home/pi/Downloads/project.zip      # 找到了！

find ~ -name "*.sh"                  # 搜所有 .sh 文件
/home/pi/pi-apps/install.sh         # 列出所有 .sh
/home/pi/pi-apps/uninstall.sh

25.5 总结：拿到路径后怎么用？

目标对你看到的路径输入的命令
进入这个文件夹/home/pi/Downloadscd /home/pi/Downloads
解压这个文件/home/pi/Downloads/project.zipcd /home/pi/Downloads 再 unzip project.zip
运行这个 .sh/home/pi/pi-apps/install.shbash /home/pi/pi-apps/install.sh
查看这个文件/home/pi/Desktop/123.txtcat /home/pi/Desktop/123.txt
💡 记住一个口诀：先 pwd 看在哪 → 再 realpath 看文件在哪 → 用 cd 跳过去 → 然后操作！三步不出错。

27. 安装代理 App（科学上网）

树莓派上装代理，可以让你在终端里流畅地 git clone、wget、apt update 访问外网资源。下面教你装三种最常见的方案。

💡 提醒：代理需要你自己有节点（订阅链接），本教程不提供节点，只教怎么装客户端。

27.1 方案一：安装 v2rayA（网页管理，最推荐新手）

v2rayA 是一个网页版代理客户端，装好后浏览器访问就能管理，支持 v2ray/trojan/ss 等协议。

1 安装 v2ray 内核 + v2rayA
# 安装 v2ray 内核
sudo apt install curl -y
bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

# 安装 v2rayA
wget https://github.com/v2rayA/v2rayA/releases/latest/download/v2raya_linux_arm64_1.5.9.5.deb
sudo dpkg -i v2raya_linux_arm64_*.deb
sudo apt install -f -y
2 启动 v2rayA
sudo systemctl enable v2raya --now
3 打开浏览器，访问 http://树莓派IP:2017
  • 创建管理员账号
  • 点 导入 → 粘贴你的订阅链接或 vmess 链接
  • 选一个节点 → 点 连接
  • 代理模式选 大陆白名单（只代理国外网站，国内网站直连）
💡 v2rayA 最方便的地方：设置好之后终端里 git clone、pip install、apt update 都会自动走代理，不需要额外配置！

27.2 方案二：安装 Clash for Windows（命令行版）

如果你用的是 Clash 订阅链接，可以用这个轻量版：

# 下载 Clash（ARM 版）
wget https://github.com/Dreamacro/clash/releases/latest/download/clash-linux-arm64-latest.gz
gunzip clash-linux-arm64-latest.gz
chmod +x clash-linux-arm64-latest
sudo mv clash-linux-arm64-latest /usr/local/bin/clash

# 创建配置文件夹
mkdir -p ~/.config/clash
cd ~/.config/clash

把你的 config.yaml（订阅文件）放到 ~/.config/clash/ 里，然后：

clash    # 启动 Clash（默认监听 7890 端口）

然后在终端里设置环境变量就能用了：

export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890

27.4 让终端总是走代理（不用每次都输 export）

把这两行加到 ~/.bashrc 文件里，每次打开终端就自动设置好代理：

echo 'export http_proxy=http://127.0.0.1:2017' >> ~/.bashrc
echo 'export https_proxy=http://127.0.0.1:2017' >> ~/.bashrc
source ~/.bashrc    # 立即生效
⚠️ 端口号要对上：上面写的是 2017（v2rayA 的端口），如果你用的是 Clash 就改成 7890。
